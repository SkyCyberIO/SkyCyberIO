# README

## 关于 SkyCyberIO

SkyCyberIO 是一次尝试，我们想试着把整个公司都开源出来，是的你没有看错，除了涉及客户数据的内容，我们都打算全部进行开源

对于 SkyCyberIO 本身，我们是一家提供 SaaS 软件定制开发的团队，服务中小微企业及团队

## 如何加入

SkyCyberIO 是一个开源组织，我们欢迎任何人加入这个团队，加入后您将获得：

- 完备的知识储备分享
- 孵化自己的项目，并获得专业指导
- 将自己的项目上线并落地，获得商业回报

如您对加入 SkyCyberIO 感兴趣，欢迎在 [Discussions](https://github.com/orgs/SkyCyberIO/discussions) 中讨论咨询
