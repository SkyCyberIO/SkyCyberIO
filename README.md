# SkyCyberIO

*写给第一个 commit*

一直以来我都有创办企业的想法，究其原因是因为我作为个人“太脆弱了”，我时刻警惕自己作为一个“个人”所带来的风险。创办一家企业并把我了解到的知识进行记录、分享，这个是我创办企业的初衷，也正是 **SkyCyberIO**

SkyCyberIO 取名很简单，“Sky” 来自公司中文名；“Cyber” 来自主要运营业务范围，“IO” 则纯属是个特别的想法，因为实在没有好的主意，就购买了 .io 域名后缀，组合在一起后还挺顺眼的

我本人以及所有参与本项目的朋友，请允许我在这里自私的表明 SkyCyberIO 的立场：

- 我们绝不做危害社会、组织、团队的事情，对任何事情保持中立态度
- 我们要对参与本项目的朋友、使用本项目产品或服务的朋友保持友善
- 我们要做自己感兴趣的事情，而不是人云亦云去挑战那些没有边界的事情
- 我们在这里的工作不是为了迎合市场需求，仅仅只是做一个产品或服务，然后取悦自己

感谢您的阅读

SkyCyberIO 敬上
